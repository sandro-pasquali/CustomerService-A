Using Data Sources

ActiveWidgets grid can display data from the variety of data sources. The grid interacts with the external data sources through the data model object. The grid itself has simple built-in data model and you can modify its built-in model behavior by overloading the grid methods. Or you can create an external data model and attach it to the grid.
